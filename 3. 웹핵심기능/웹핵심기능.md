9. Slide-show
 - setTimeout(callback, milis),
    - callback : 실행할 함수
    - milis : 실행 대기시간(1/1000초)

    - 설정한 밀리초 시간 이후에 callback을 단 1회 실행함.

    - callback : 실행할 함수
    - milis : 실행 간격(1/1000초)

    - 설정한 밀리초 간격마다 반복해서! 지속적으로! callback을 실행한다.
